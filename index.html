Read The Shot

The client: Beach Volleyball Community

Purpose: A game designed so users can improve their ability to read volleyball shots in real time via video clips. The Volleyball Shot Reading Trainer is built on React Functional Components, employing the useState hook to manage the application's dynamic state, from controlling UI visibility and tracking the game's difficulty level (gameLevel) to updating the user's score and feedback messages. Lifecycle events and interactions with external systems are handled by the useEffect hook, responsible for dynamically fetching video details, initializing the shuffled playback queue, and managing time-sensitive logic like the blackout countdown timer in the Advanced mode. Seamless control over video playback and sound effects is achieved through the useRef hook, which provides persistent access to the YouTube Player iframe API methods and HTML Audio elements. Finally, the useMemo hook ensures performance by stably defining and caching large, static data sets, such as the initial list of video clips and arrays of possible feedback messages, contributing to a smooth and responsive training experience.
Target Audience: Sports Improvement App Users
Tools: React, Javascript, HTML, CSS, Flexbox, iframe

Todo or Not Todo

Purpose: A todo list as a dynamic web application. It uses the browser's Local Storage to provide persistent data, allowing users to create and manage multiple to-do lists and their corresponding tasks. Creating and appending HTML elements, allowing for an interactive and responsive user experience. CRUD Functionality (Create, Read, Update, Delete). Each list item functions as a container for its own set of tasks. The application includes a checkbox feature for tasks, which automatically updates the parent list's visual state (e.g., adding a "completed" line-through) when all tasks within it are marked as done. This demonstrates complex conditional logic and data synchronization. Client-side JavaScript, DOM manipulation, event handling, and modern web storage techniques to build a robust and functional application from the ground up.
